## Known Issues & Workarounds

### CSS Caching Issue
Due to aggressive browser caching, the stylesheet may not reload properly after updates. The application includes an automatic workaround that injects critical CSS via JavaScript to ensure correct layout regardless of cache state.

If you experience layout issues:
1. Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)
2. Try opening in an incognito/private window
3. Clear browser cache for localhost
4. Restart the Python server

The JavaScript-injected CSS ensures the sidebar stays fixed at 300px width and buttons are properly styled even if the cached `style.css` is stale.
